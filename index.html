My DepEd Internship Journey

A 10-week progression architecting and deploying a highly secure, dynamic web-based administrative system for the Department of Education - Division of Zamboanga City (ICT Unit).

Week 1

Team Onboarding & Database Architecture

Collaborated with the ICT team to set up our development environments and establish the foundational UI/UX. During this week, we successfully engineered the initial MySQL database schema and developed the global navigation system to ensure seamless UI integration.

Week 2

Project Deployment & User Management

Officially deployed the initial system framework with the division staff. Transitioned to a functional web app by implementing full CRUD capabilities for official memorandums and initiating backend logic for secure, role-based user access.

Week 3

Procurement Module & Curriculum Logic

Developed the dynamic Procurement page to display bid opportunities programmatically. Additionally, engineered complete CRUD functionality for the Learning Materials section and built a dynamic layout for the K-12 Basic Curriculum Guide.

Week 4

Organizational Structure & Admin Controls

Completed the "About" module, successfully mapping and rendering the department's complex Organizational Structure. Integrated CKEditor and a global configuration module, giving super-admins the power to format pages directly from the dashboard.

Week 5

Maintenance Modes & Layout Constraints

Engineered a page deactivation feature that safely triggers a custom 'We'll Be Right Back!' maintenance screen for lifecycle management. We also resolved complex responsive UI bugs related to scaling external media embeds like Facebook videos.

Week 6

Modal Standardization & Breadcrumbs

Standardized the UI/UX design of all administrative modal pop-ups, such as the "Edit Record" interface. Implemented global breadcrumbs to improve site navigability and restructured backend table layouts to ensure uniform consistency.

Week 7

Data Validation & Advanced Security

Strengthened data integrity by programming strict duplication logic across file upload modules to prevent redundant data entry, such as duplicate carousel images. Led the integration of a custom Role-Based Access Control (RBAC) mechanism and a password recovery pipeline.

Week 8

Nested Hierarchies & Data Filtering

Rewrote the dynamic pages module using recursive database structures to support deeply nested, multi-level dropdown navigation menus. Engineered multi-layered filtering for student directories and unified upload workflows to eliminate repetitive data entry.

Week 9

Interactive Map API Integration

Collaborated across development teams to securely fetch and integrate an external API using Postman, successfully rendering a dynamic interactive map of the region. We also finalized the Procurement data module with advanced date filtering.

Week 10

Final Presentation & System Refinement

Presented the comprehensive, finalized system to the ICT supervisor and team for evaluation in the conference room. Addressed user acceptance feedback, stabilized the home banner, and finalized the admin login interface for official deployment.
